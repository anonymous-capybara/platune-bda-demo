# Leveraging Small Datasets for Generative Modelling with Musical Controls

Demo page for the paper *"Leveraging Small Datasets for Generative Modelling with Musical Controls"*.

**Live site:** [https://anonymous-capybara.github.io/platune-bda-demo/](https://anonymous-capybara.github.io/platune-bda-demo/)

## Local Development

Prerequisites: Ruby and Bundler installed.

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve
```

Then open [http://localhost:4000](http://localhost:4000).
